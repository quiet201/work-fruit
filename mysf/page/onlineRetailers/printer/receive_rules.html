    活动规则
    常见问题解答

  • 　活动规则

    　　本活动仅限顺丰月结客户参加

    　　如与顺丰终止合作，顺丰有权将打印机回收

    　　本活动一切解释权归顺丰所有

  • 　活动说明

    　　点击并支付成功1000元申领金，即可获取热敏打印机一台

    　　申领金将转入顺手付理财账户，并进行冻结

    　　顺丰将每月进行统计，待客户发件量满600票后可将申领金进行解冻

    　　解冻后，用户即可自由进行申领金的提取和处置

  • Q :　我如何才能看到我的申领金？

    A :　您可以点击APP首页中”理财“图标，进入查看.

  • Q :　我的申领金支付成功后什么时候可以获取打印机？

    A :　您的申领金支付成功后，顺丰将在7个工作日内安排人员与您联系，并安排打印机和安装，如您有特别要求，也可届时再联系电话中说明。 　　

  • Q :　我的申领金冻结后是否还可以提现？

    A : 　　您的申领金进入理财冻结后，将无法进行转出和提现，但仍可享受理财收益。

    　　　　需待您的申领金解冻后，方可转出。

  • Q :　申领金冻结期间的利息如何进行计算？

    A :　您的申领金从支付成功后的第二个交易开始计算收益，遇节假日顺延。具体收益率以理财基金当日公布的收益为准。　　

  • Q :　要满足什么条件才能将申领金解冻？

    A : 　　商家需使用顺丰电商产品累计发件量满600票（以申领成功后开始计算），方可进行解冻。

    　　　　发件量以自然月度进行统计，在每月7日前完成名单审核并进行解冻。每月只进行一次批量解冻。

    　　　　如您业务量未达要求但仍需要解冻，可申请“提前解冻”。

  • Q :　我如果要提前申请解冻如何处理？

    A : 　　申领金的解冻原则上将由顺丰根据商家业务量来进行主动解冻。如您业务量未达要求仍需解冻，则可申请提前解冻，但解冻成功后，打印机将由顺丰进行收回。

    　　　　提前解冻可通过以下方式申请：

    　　　　电话拨打4000 555 555，阐明解冻理由，确认后由顺丰安排解冻。

  • Q :　我如何知道我的申领金是否已解冻？

    A :　申领金解冻后，将在2个工作日内进行短信告知。

  • Q :　申领金解冻后是否可以提现？

    A :　您的申领金一旦解冻后即可自由操作。您可以选择继续理财或转出提现至银行卡。
